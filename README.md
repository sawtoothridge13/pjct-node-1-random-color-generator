# Node.js Random Color Generator

A ransom color generator CLI app that will be able to generate completely random colors:

```bash
$ node index.js
###############################
###############################
###############################
#####                     #####
#####       #ff0201       #####
#####                     #####
###############################
###############################
###############################
```

Or random colors based on input:

```bash
$ node index.js
###############################
###############################
###############################
#####                     #####
#####       #ff0200       #####
#####                     #####
###############################
###############################
###############################
```
